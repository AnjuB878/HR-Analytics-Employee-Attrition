# HR Analytics: Employee Attrition Analysis

## 📊 Project Overview

This project analyzes employee attrition to identify key factors influencing employee turnover and provide data-driven insights for HR decision-making.

---

## 🎯 Objectives

* Analyze employee data to understand attrition patterns
* Identify key drivers of employee turnover
* Build an interactive dashboard for HR insights

---

## 🛠 Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* Power BI
* SQL
* Excel

---

## 📂 Dataset

IBM HR Analytics Employee Attrition Dataset

---

## 🔍 Key Analysis

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Attrition trend analysis
* Feature impact analysis

---

## 📊 Dashboard Highlights

* Attrition Rate KPI
* Attrition by Department
* Attrition by Salary Band
* Attrition vs Overtime
* Attrition by Job Satisfaction
* Attrition by Years at Company

---

## 🧠 Key Insights

* Employees working overtime are more likely to leave
* Lower salary bands show higher attrition
* Most attrition occurs within the first 2–3 years
* Job satisfaction significantly impacts retention

---

## 📈 Outcome

This project demonstrates how data analytics can support HR teams in reducing attrition and improving employee retention strategies.
